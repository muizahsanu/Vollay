# Vollay

Vollay is a simple Android utility app that allows you to control system volume using a floating overlay button.

This app was created for personal use as a workaround for devices with damaged or unreliable physical volume buttons.

## Features
- Floating overlay button
- System volume control
- Works across all applications
- No root required

## Technical Overview
This project explores several core Android concepts, including:
- Foreground services
- System overlay using WindowManager
- Touch gesture handling
- AudioManager integration

## Use Case
Designed as an alternative way to adjust volume when hardware volume buttons are not available.

## Status
Personal project / Experimental  
Built primarily for learning and daily personal use.
